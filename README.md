# BlazePose Graduation Project

Unity Sentis 기반 BlazePose 포즈 인식 졸업 프로젝트입니다.

웹캠에서 사람의 자세를 실시간으로 추적하고, 추적된 관절 좌표를 인터랙티브 콘텐츠에 연결하는 기능을 개발하고 있습니다.

## Project Focus

- BlazePose를 이용한 실시간 포즈 감지
- Unity 환경에서 포즈 좌표 시각화
- 인식 결과와 몸의 움직임을 활용한 상호작용 구현
- 졸업 프로젝트를 위한 프로토타입 제작
